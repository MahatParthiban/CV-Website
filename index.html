Investigating the Barocaloric Effect in DABCO Oxalate Crystals Crystallography X-Ray Diffraction Calorimetry Material Science Python

A lab-based, practical investigation into the barocaloric properties of DABCO Oxalate for use in a solid-state barocaloric-effect refridgerator

  • DABCO Oxalate crystals were sucessfully synthesized using solvent layering and slow evaporation.
  • Crystal structure was identified using powder X-ray diffraction and single crystal X-ray diffraction.
  • High-Pressure Differential Scanning Calorimetry (HPDSC) equipment was used to explore the barocaloric response of the crystals under different pressures.
  • Calorimetry data was analysed using python

    • Analysis of the Lomb-Scargle and cubic interpolation methods for use on irregularly sampled voltage data Python Data Analysis Scientific Computing

    This project focused on the concept of power system harmonics, using the Fast Fourier Transform (FFT) to identify significant harmonic components of synthetic and real-world power system voltage data.

    • A synthetic dataset was created using Numpy and Pandas by overlaying random noise onto a sinusoidal voltage signal.
    • Python functions were created based on various irregular sampling analysis methods such as the Lomb-Scargle and cubic interpolation
    • Functions were tested using a real power system dataset with the obtained values compared against power quality standards.
